# View

### Input

- [ ] 메뉴 추천을 받을 코치 이름을 입력받는다.
    - 2명에서 5명까지
- [ ] 각 코치가 못먹는 메뉴를 입력받는다.
    - 최대 2개까지

### Output

- [ ] 입력에 대한 안내문구
- [ ] 메뉴 추천 결과
- [ ] 예외에 따른 문구
